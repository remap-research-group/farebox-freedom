This repository contains all data files and scripts for the paper titled '**_Farebox Freedom: Exploring Fare Cost Burdens Relative to the Suburbanization of Poverty in the United States_**' authored by Vir Chachra, Rounaq Basu, and Jim Aloisi.

- **factor_analysis.R** contains the steps for the Exploratory Factor Analysis leading to the estimation of three sociodemographic factors.	
- **models_w_factor_analysis.R** contains the model estimation process.
- **write_models.R** writes the results of the models estimated above to output files.

- **full_data_final.zip** contains all necessary data files. All data are open-source and can be shared publicly with proper attribution.
